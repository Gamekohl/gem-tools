Here you can find all important keybindings for the GEM-Editor.

## General

| Keybinding      | Description        |
|-----------------|--------------------|
| F1              | Game Interface     |
| F2              | Map Menu           |
| F3              | Mission Menu       |
| F6              | Environment        |
| F9 (in F3 mode) | Mission Properties |

## F2 - Map Menu

| Keybinding | Description     |
|------------|-----------------|
| 1          | Entities        |
| 2          | Terrain         |
| 3          | Camera (Global) |
| 4          | Edifice         |

## F3 - Mission Menu

| Keybinding | Description        |
|------------|--------------------|
| 1          | View               |
| 2          | Entities           |
| 3          | Squads             |
| 4          | Waypoints          |
| 5          | Zones              |
| 6          | Triggers           |
| 7          | Camera (Cinematic) |
| 8          | Covers             |

## CTRL +

| Keybinding | Description            |
|------------|------------------------|
| Q          | Camera Lock            |
| T          | Add Tags               |
| I          | Inventory modification |
| O          | Open                   |
| S          | Save                   |
| F          | Find Entity            |
| 0          | Center on Map          |
| M          | Map                    |

## ALT +

| Keybinding | Description                           |
|------------|---------------------------------------|
| W          | Polygon view of terrain and objects   |
| R          | Call Functions                        |
| T          | Show Tags                             |
| I          | Show MIDS                             |
| S          | Show XYZ Coordinates of Entities      |
| H          | Show Heightmap                        |
| V          | Show Entity Polygons                  |
| X          | Shut down Editor                      |
| A          | Switch Obstacles on and off           |
| P          | Switch AI Chosen Waypoints on and off |
| Z          | Undo/Redo Window                      |
| F          | Toggle Fog of War                     |

## SHIFT +

| Keybinding | Description       |
|------------|-------------------|
| T          | Bring up Textures |
| F          | First Person View |

## Single Letters

| Keybinding | Description               |
|------------|---------------------------|
| C (hold)   | Rotate Entity             |
| A (hold)   | Change Altitude of Entity |
| S (hold)   | Change Scale of Entity    |

## Misc

| Keybinding        | Description                         |
|-------------------|-------------------------------------|
| CTRL-ALT-LSHIFT-D | Hide UI                             |
| CTRL-ALT-D        | Hide UI, except Map and Player Side |
